moment4j
========

Java API for the moment.js library


SNAPSHOT:

```xml
<repositories>
  <repository>
  <id>sonatype</id>
  <url>https://oss.sonatype.org/content/repositories/snapshots/</url>
  </repository>
</repositories>

<dependency>
  <groupId>com.ahome-it</groupId>
  <artifactId>moment4j</artifactId>
  <version>1.0.0-SNAPSHOT</version>
</dependency>
```
